Lab 1: Word Frequency Counter Using Multithreading

Programming language: Python 3

Files included:
- word_frequency_counter.py: main source code
- sample_input.txt: sample text file for testing
- Lab1_Report.docx: short report explaining the design and sample output
- README.txt: instructions to run the program

How to run:
1. Make sure Python 3 is installed.
2. Open a terminal or command prompt in this folder.
3. Run the program using this format:

   python word_frequency_counter.py sample_input.txt 3

The first argument is the text file name.
The second argument is the number of segments/threads.

Example:
   python word_frequency_counter.py sample_input.txt 3

The program will print:
- the intermediate word-frequency count from each thread
- the final consolidated word-frequency count
